# Quotes Generator

## Description
- Generates quotes from [typefit]('https://typefit.com")
- Displays it on the DOM and fully responsive
- Connects with twitter using [twitter-intent]('https://twitter.com/intent')

## Technologies Used
HTML5 || CSS || VanillaJS
